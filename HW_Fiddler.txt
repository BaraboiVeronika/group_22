Ex_2:
Method: POST
EndPoint: /user_info_3
request form data: 
 name: str
 age: int
 salary: int

response: 
{'name': name,
          'age': age,
          'salary': salary,
          'family': {'children': [['Alex', 24], ['Kate', 12]],
                     'u_salary_1_5_year': salary * 4}}

Task:
Сделать правила:
 ⁃ Подменить тело запроса чтобы поменялся name которые вы вписали в Postman.
 - Подменить тело запроса чтобы поменялся age которые вы вписали в Postman. 
 - Подменить тело запроса чтобы поменялся salary которые вы вписали в Postman. 
 - Подменить тело запроса чтобы удалился age которые вы вписали в Postman.  (Получить 500 код)
 - В ответе поменять children на neighbors. 
 - В ответе поменять значение зарплаты u_salary_1_5_year на другую цифру. 
 - В ответе удалить параметр salary. 

Действия: 
1) Направить POST-запрос через Postman. Выбрать этот запрос в Fiddler. Перейти во вкладку Rule -> 
Add New Rule. В части Conditions выбираем URL -> Contains -> вставляем URL из запроса. В части Actions
выбираем Update Request Body -> Find and replace -> Gosha заменить на Natalia. 
2) Нажать на Add action и добавить аналогично замену age c 25 на 33. 
3) Нажать на Add action и добавить аналогично замену salary c 500 на 1400. 
4) Перейти во вкладку Rule -> Add New Rule. В части Conditions выбираем URL -> Contains -> вставляем URL из запроса. 
В части Actions выбираем Update Request Body -> Find and replace -> 25 заменить на пустое поле. 
5) Перейти во вкладку Rule -> Add New Rule. В части Conditions выбираем URL -> Contains -> вставляем URL из запроса. 
В части Actions выбираем Update Response Body ->  Find and replace -> children заменить на neighbors.
6) Нажать на Add action и добавить аналогично замену u_salary_1_5_year с 2000 на 500.
7) Перейти во вкладку Rule -> Add New Rule. В части Conditions выбираем URL -> Contains -> вставляем URL из запроса. 
В части Actions выбираем Update Response Body -> Find and replace -> salary на пустое значение.

                                     'age': 4}},
                     'u_salary_1_5_year': salary * 4}
          }

Task:
Сделать правила:
 ⁃ Подменить url в чтобы в запросе поменялся name которые вы вписали в Postman.

В части Actions выбираем Update Response Body -> Find and replace -> salary на пустое место.
5) Перейти во вкладку Rule -> Add New Rule. В части Conditions выбираем URL -> Contains -> вставляем URL из запроса. 
В части Actions выбираем Manual Response -> изменяем json cat на {"count":2,"host_name":"Nikita"}.
6)  Перейти во вкладку Rule -> Add New Rule. В части Conditions выбираем URL -> Contains -> вставляем URL из запроса. 
В части Actions выбираем Manual Response -> изменяем статус код 200 ОК на 405 METHOD NOT ALLOWED.

Ex_6:
Method: POST
EndPoint: /user_info_2
request form data: 
 name: str

response: 
{'start_qa_salary': salary,
          'qa_salary_after_6_months': salary * 2,
          'qa_salary_after_12_months': salary * 2.7,
          'qa_salary_after_1.5_year': salary * 3.3,
          'qa_salary_after_3.5_years': salary * 3.8,
          'person': {'u_name': [user_name, salary, age],
                     'u_age': age,
                     'u_salary_5_years': salary * 4.2}
          }


Task:
Сделать правила:
 - Подменить тело запроса чтобы поменялся age которые вы вписали в Postman. 
 - Подменить тело запроса чтобы поменялся salary которые вы вписали в Postman. 
 - Подменить тело запроса чтобы удалился salary которые вы вписали в Postman.
 - В ответе поменять qa_salary_after_6_months на qa_salary_after_10_months. 
 - В ответе поменять значение зарплаты qa_salary_after_1.5_year на другую цифру. 
 - В ответе удалить параметр person. 
 - В ответе поменять значение параметр person с json на xml. 

Действия: 
1) Направить POST-запрос через Postman. Выбрать этот запрос в Fiddler. Перейти во вкладку Rule -> 
Add New Rule. В части Conditions выбираем URL -> Contains -> вставляем URL из запроса. В части Actions
выбираем Update Request Body -> Find and replace -> 40 заменить на 30. 
2) Нажать на Add action и добавить аналогично замену salary c 4000 на 3000. 
3) Перейти во вкладку Rule -> Add New Rule. В части Conditions выбираем URL -> Contains -> вставляем URL из запроса. 
В части Actions выбираем Update Request Body -> Find and replace -> 4000 заменить на пустое поле. 
4) Перейти во вкладку Rule -> Add New Rule. В части Conditions выбираем URL -> Contains -> вставляем URL из запроса. 
В части Actions выбираем Update Response Body ->  Find and replace -> qa_salary_after_6_months на qa_salary_after_10_months. 
5) Нажать на Add action и добавить аналогично замену qa_salary_after_1.5_year с 13200.0 на 20000.
6) Нажать на Add action и добавить аналогично замену person на пустое значение. 
7) Перейти во вкладку Rule -> Add New Rule. В части Conditions выбираем URL -> Contains -> вставляем URL из запроса. 
В части Actions выбираем Manual Response -> изменяем параметр person с json на xml:
